## How to Run Locally

### 1. Clone repository
```bash
git clone https://github.com/your-repo/warehouse-management.git
cd warehouse-management
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run development server
```bash
npm run dev
```
Access at: http://localhost:3000

### 4. Run tests
```bash
npm run test
```

### 5. Build for production
```bash
npm run build
```
